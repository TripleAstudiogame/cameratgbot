# Установка и запуск MailToTelegram Bot

Этот проект предназначен для автоматической пересылки писем с камер видеонаблюдения (NVR) в Telegram. Проект адаптирован для работы на Windows Server 2019 в качестве фоновой задачи.

## 🛠 Масштабируемость
Чтобы запустить несколько ботов для разных объектов или почтовых ящиков:
1. Просто скопируйте всю папку проекта в новое место (например, `MailToTelegram_Object2`).
2. В новой папке измените настройки в файле `.env` (укажите другой токен бота и/или другую почту).
3. Запустите для каждой папки свой `install_task.ps1` с новым именем задачи.

## 🚀 Как развернуть на новом сервере (Windows Server)

### Шаг 1. Установка зависимостей
1. Убедитесь, что на сервере установлен **Python 3.9+**. При установке Python обязательно поставьте галочку **"Add Python to PATH"**.
2. Откройте папку проекта и запустите файл `install.bat`. 
   Он автоматически создаст виртуальное окружение и установит все нужные библиотеки из `requirements.txt`.

### Шаг 2. Настройка конфигурации
1. Откройте файл `.env` любым текстовым редактором (например, Блокнотом).
2. Укажите актуальные данные:
   - `TELEGRAM_BOT_TOKEN` — токен вашего бота (от BotFather).
   - `MAIL_USERNAME` — почта, с которой нужно забирать письма.
   - `MAIL_APP_PASSWORD` — пароль приложения для почты (не обычный пароль от аккаунта!).

### Шаг 3. Запуск в фоне (Автозагрузка)
Чтобы бот работал круглосуточно, не требовал открытого черного окна консоли и сам запускался при перезагрузке сервера:

1. Нажмите правой кнопкой мыши на файл `install_task.ps1` и выберите **"Run with PowerShell"** (Выполнить с помощью PowerShell).
2. Скрипт создаст невидимую задачу в планировщике заданий Windows (Task Scheduler).
3. Бот будет автоматически запускаться от имени системы (SYSTEM) при каждом запуске сервера.

### Как остановить бота или проверить логи
- **Логи:** Вся информация об ошибках и работе пишется в файл `bot.log`. Просто откройте его блокнотом.
- **Остановка:** Откройте «Планировщик заданий» (Task Scheduler) в Windows, найдите задачу `MailToTelegramBot`, нажмите по ней правой кнопкой и выберите `End` (Завершить) или `Disable` (Отключить).
